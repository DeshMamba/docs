The Tag Contact action applies a label to the contact in Wave Runner.

## Configuration

| Field | Required | Description |
| --- | --- | --- |
| **Tag** | Yes | The label to apply to the contact |

## Use case

A Condition node detects that the call outcome is `interested`. The Tag Contact action applies the tag "interested" to the contact. That tag is then used to filter a follow-up campaign list, so only tagged contacts get the next outbound sequence.
