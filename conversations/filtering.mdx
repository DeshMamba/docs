Use the toolbar at the top of Conversations to find what you need.

Filters work together. Combine any of them to narrow the list.

## Search

Type a name, phone number, or keyword. Chattrvox matches against contact names, phone numbers, call summaries, and outcomes.

## Date range

Set a start and end date to show only contacts with activity in that window.

## Outcome

Filter by how the most recent call ended:

- Booked Appointment
- Qualified
- Not Qualified
- Spam
- Escalated
- Call Dropped

## Agent

Show only conversations handled by a specific AI assistant. Useful when you run multiple assistants across different use cases.

## Channel

- **Phone** — standard calls via Twilio
- **Web Voice** — calls from an embedded website widget
